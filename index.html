🔧  Setup — do this first on every laptop
1. Install VS Code
Free code editor. Download the installer, run it, done.
↗ code.visualstudio.com
2. Install Node.js
Click the big green LTS button. Installs npm too.
↗ nodejs.org
3. Install GitHub Desktop
How you save and share your code with your team. Download, install, sign in.
↗ desktop.github.com
4. Install Cline (AI helper)
Open VS Code → click the Extensions icon (4 squares, left sidebar) → search Cline → Install.
